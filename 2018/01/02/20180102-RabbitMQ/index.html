RabbitMQ

server
  • 消息队列

RabbitMQ 是实现了高级消息队列协议（AMQP）的开源消息代理软件（亦称面向消息的中间件）。RabbitMQ服务器是用Erlang语言编写的，而集群和故障转移是构建在开放电信平台框架上的。所有主要的编程语言均有与代理接口通讯的客户端库。

RabbitMQ 降低主程序线程压力,是一种非阻塞模式的分布式消息队列服务器，有生产者生产到 RabbitMQ，再由消费者消费。
