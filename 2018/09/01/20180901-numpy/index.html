Numpy 学习大纲

python
  • Numpy
  • python
  • 二维数组
  • 多维数组
  • 矢量计算
  • 科学计算

NumPy系统是Python的一种开源的数值计算扩展。这种工具可用来存储和处理大型矩阵，
比Python自身的嵌套列表（nested list structure) 结构要高效的多（该结构也可以用来表示矩阵（matrix））。
